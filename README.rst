Framework For Selenium

* Free software: ISC license

Features
--------

* Runner for Amazon EC2

Quick-start
-----------

    ::

    $ cookiecutter https://github.com/brome-hq/brome
    $ chmod +x bro
    $ ./bro install
    $ vim config/brome.yml
    $ ./bro admin --create-database
