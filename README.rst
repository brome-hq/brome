Quick-start
-----------

.. role:: bash(code)
   :language: bash

    :bash:`cookiecutter https://github.com/brome-hq/brome`.
    :bash:`chmod +x bro`.
    :bash:`./bro install`.
    :bash:`vim config/brome.yml`.
    :bash:`./bro admin --create-database'`.
